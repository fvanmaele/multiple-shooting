Detailed Description

template<typename Callable>
class FAD_Setup< Callable >

Class representing the time-dependent function with components .

Both evaluation and automatic differentation (using the Sacado package from the Trilinos library) are supported.

For general information, see SESS 2007, E. Phipps:

https://software.sandia.gov/SESS/past_seminars/111307_Phipps.html
